# 📚 Library Management System using SQL

## Overview

This project is a **Library Management System** developed using **MySQL**. It is designed to manage the day-to-day operations of a library by storing and organizing information about books, members, borrowing transactions, returns, and fines.

The project demonstrates the implementation of a relational database using SQL concepts such as tables, relationships, primary keys, foreign keys, joins, views, procedures, and queries.

---

## Features

- Add and manage books
- Maintain member records
- Issue books
- Return books
- Track borrowed books
- Fine calculation
- Search books
- Generate reports
- SQL queries for library operations

---

## Technologies Used

- MySQL
- SQL
- DBMS Concepts

---

## Database Structure

The project contains tables such as:

- Books
- Members
- Authors
- Publishers
- Categories
- Book_Issue
- Book_Return
- Fine
- Staff (Optional)

These tables are connected using appropriate primary and foreign key relationships.
